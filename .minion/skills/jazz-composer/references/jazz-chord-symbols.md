# Jazz Chord Symbols Reference

## Basic Chord Types
- **maj7, Δ7, -**: Major seventh (1-3-5-7)
- **7**: Dominant seventh (1-3-5-b7)  
- **m7, -7**: Minor seventh (1-b3-5-b7)
- **m7b5, ø7**: Half-diminished (1-b3-b5-b7)
- **°7**: Fully diminished (1-b3-b5-bb7)

## Extended Chords
- **9**: Add ninth to seventh chord
- **11**: Add eleventh (often implies sus4 for dominant)
- **13**: Add thirteenth (usually omits 11th)

## Alterations
- **b9, #9**: Altered ninths
- **#11**: Sharp eleventh (Lydian sound)
- **b13**: Flat thirteenth

## Common Jazz Progressions
- **ii-V-I**: Dm7-G7-Cmaj7
- **I-vi-ii-V**: Cmaj7-Am7-Dm7-G7  
- **iii-vi-ii-V**: Em7-Am7-Dm7-G7
- **Tritone Substitution**: Db7 instead of G7 resolving to C

## Voicing Principles
- Rootless voicings for piano (omit root, emphasize 3rd and 7th)
- Drop 2 and drop 3 voicings for guitar
- Shell voicings (3rd and 7th only) for minimal accompaniment
